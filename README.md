# Plateforme-d-Orchestration-IA-pour-la-Classification-Zero-Shot-et-la-Synth-se-Contextuelle-Frontend
Hybrid-Analyzer est une application conçue pour automatiser l’analyse d’articles de veille grace à l’orchestration de deux services d’IA complementaires : Hugging Face (Zero-Shot Classification) pour identifier la catégorie d’un texte sans entrainement prealable, API Gemini pour produire un résumé contextualisé et analyser la tonalité du contenu.
